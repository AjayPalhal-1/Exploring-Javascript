Why Are Functions Called First-Class Citizens in JavaScript?

Functions in JavaScript are called first-class citizens because they can be treated just like any other
data type (e.g., numbers, strings, objects). They have the following capabilities:

1. Assignment to Variables: Functions can be assigned to variables like any other value.
2. Passed as Arguments: Functions can be passed as arguments to other functions.
3. Returned from Functions: Functions can return other functions as their output.
4. Stored in Data Structures: Functions can be added to arrays, objects, or other data
   structures.
